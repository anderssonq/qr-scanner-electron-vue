# vue-electron-js

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```
run in web dev

```
yarn electron:serve
```
run in desktop dev

yarn build or yarn electron:build will create a production app

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
